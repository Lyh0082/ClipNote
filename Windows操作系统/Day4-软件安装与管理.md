# 📦 Day 4：软件安装与管理

> **学习目标**：学会安装和卸载软件、了解常用软件推荐、掌握软件更新方法、认识绿色软件和安装版的区别。

---

## 1️⃣ 软件的两种类型

| 类型 | 特点 | 举例 |
|------|------|------|
| **安装版** | 有安装向导，写入注册表，可在"程序和功能"中卸载 | QQ、微信、Office |
| **绿色版/便携版** | 解压即用，不写注册表，删除文件夹即可卸载 | Notepad++便携版 |

> 💡 推荐新手优先使用安装版，管理更规范。

---

## 2️⃣ 安装软件的三种方式

### 方式一：Microsoft Store（微软商店）
1. 点击开始菜单 → 打开 **Microsoft Store**
2. 搜索你想要的软件
3. 点击 **获取/安装**
4. 自动下载安装

优点：安全可靠、自动更新
缺点：软件种类有限

### 方式二：官网下载安装包
1. 用浏览器访问软件的**官方网站**（注意认准官网！）
2. 下载安装包（通常是 `.exe` 或 `.msi` 文件）
3. 双击安装包，跟随安装向导
4. ⚠️ 注意看清每一步，取消勾选**捆绑软件**

### 方式三：应用包管理器（进阶）
- 使用 `winget`（Windows自带）或 `scoop` 安装
- 示例：打开终端输入 `winget install WeChat`
- 优点：快速、批量安装
- 本课程暂时不深入，了解即可

---

## 3️⃣ 安装注意事项

### ⚠️ 常见陷阱

1. **下载站陷阱**：不要从非官方网站下载，容易带捆绑软件
   - 推荐下载站：软件官网 > 腾讯软件中心 > 少数派
   - 避免的网站：各种"下载站"、带有"高速下载"按钮的网站

2. **捆绑安装**：安装过程中注意取消勾选额外软件
   - 常见捆绑：浏览器主页锁定、杀毒软件、各种助手
   - 如果不确定，选 **自定义安装** 而不是快速安装

3. **安装路径**：
   - 默认装在C盘即可（现代电脑C盘够大）
   - 如果C盘空间紧张，可以装到D盘
   - ⚠️ 不要把软件装在桌面或中文路径的深层文件夹里

## 4️⃣ 卸载软件

### 方法一：设置卸载（推荐）
1. `Win + I` 打开设置 → **应用** → **已安装的应用**
2. 找到要卸载的软件 → 点击 **三个点** → **卸载**
3. 按提示完成卸载

### 方法二：控制面板卸载
1. `Win + S` 搜索"控制面板"
2. 点击 **程序** → **程序和功能**
3. 右键要卸载的软件 → **卸载**

### 方法三：开始菜单卸载
1. 打开开始菜单 → 找到软件
2. 右键 → **卸载**

> ⚠️ 不要直接删除文件夹来"卸载"软件！这会留下注册表残留。一定要走正规卸载流程。

---

## 5️⃣ 软件更新

### Microsoft Store安装的软件
- 自动更新（可关闭）
- 手动检查：打开 Microsoft Store → 点击库图标 → 获取更新

### 官网安装的软件
- 大多数软件有内置更新机制
- 打开软件 → 找到"帮助"或"设置" → "检查更新"
- 或直接去官网下载最新版覆盖安装

## 6️⃣ 新手必备软件推荐

| 类别 | 推荐软件 | 说明 |
|------|----------|------|
| 浏览器 | Microsoft Edge | 系统自带，够用 |
| 办公套件 | WPS Office | 免费且兼容性好 |
| 压缩工具 | 7-Zip | 免费开源，无广告 |
| 即时通讯 | 微信/QQ | 国内常用 |
| 视频播放 | VLC | 支持几乎所有格式 |
| 截图工具 | Snipaste | 专业截图+标注 |
| PDF阅读 | SumatraPDF | 轻量快速 |
| 安全防护 | Windows Defender | 系统自带，无需额外安装 |
