# ⚡ Day 7：效率工具与综合实战

> **学习目标**：掌握Windows内置效率工具、学会更多快捷键、完成一个综合实战项目、回顾七天学习内容。

## 1️⃣ 效率神器：快捷键大全

### 系统级快捷键（最常用）

| 快捷键 | 功能 |
|--------|------|
| `Win + D` | 显示桌面 / 恢复窗口 |
| `Win + E` | 打开文件资源管理器 |
| `Win + I` | 打开设置 |
| `Win + L` | 锁定屏幕（离开电脑时用！） |
| `Win + S` | 搜索 |
| `Win + V` | 剪贴板历史（超实用！） |
| `Win + Tab` | 任务视图（所有打开的窗口） |
| `Win + , `| 临时预览桌面（松开恢复） |
| `Win + ↑` | 最大化当前窗口 |
| `Win + ↓` | 最小化/还原当前窗口 |
| `Win + ←` | 窗口靠左半屏 |
| `Win + →` | 窗口靠右半屏 |
| `Alt + Tab` | 切换窗口 |
| `Alt + F4` | 关闭当前窗口/关机 |

### 文件操作快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + C` | 复制 |
| `Ctrl + X` | 剪切 |
| `Ctrl + V` | 粘贴 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + Y` | 重做 |
| `Ctrl + A` | 全选 |
| `F2` | 重命名 |
| `Delete` | 删除（进回收站） |
| `Shift + Delete` | 永久删除 |

## 2️⃣ 虚拟桌面

虚拟桌面让你拥有多个桌面，像有多台显示器一样：

### 创建和切换
| 操作 | 方法 |
|------|------|
| 打开任务视图 | `Win + Tab` |
| 新建桌面 | 任务视图中点 **+ 新建桌面** |
| 切换桌面 | `Win + Ctrl + ←/→` |
| 关闭桌面 | 任务视图中悬停桌面 → 点 ✖ |

> 💡 使用场景：学习放一个桌面、娱乐放一个桌面，互不干扰。

## 3️⃣ 内置效率工具

### 截图工具
| 快捷键 | 功能 |
|--------|------|
| `Win + Shift + S` | 截图（推荐！） |
| `Print Screen` | 截取全屏到剪贴板 |
| `Alt + Print Screen` | 截取当前窗口 |

截图后可以选择：矩形截图 / 自由截图 / 窗口截图 / 全屏截图

### 剪贴板历史
1. 按 `Win + V` 打开剪贴板历史
2. 可以看到最近复制的所有内容
3. 点击即可粘贴之前的内容
4. 首次使用需要先开启：设置 → 系统 → 剪贴板 → 开启

### 录屏工具
- `Win + G` 打开Xbox Game Bar
- 点击录制按钮即可录制屏幕
- 适合录制操作教程或游戏

### 便笺（Sticky Notes）
- `Win + S` 搜索"便笺"
- 类似桌面便利贴，随手记笔记
- 支持多种颜色区分

### 计算器
- `Win + S` 搜索"计算器"
- 标准/科学/程序员/日期计算多种模式

### 画图
- `Win + S` 搜索"画图"
- 简单的图片编辑工具

## 4️⃣ 文件搜索进阶

### 搜索语法
在资源管理器搜索框中使用：

| 语法 | 示例 | 说明 |
|------|------|------|
| `*` 通配符 | `*.jpg` | 所有JPG文件 |
| `kind:` | `kind:document` | 所有文档 |
| `date:` | `date:this week` | 本周的文件 |
| `size:` | `size:>1GB` | 大于1GB的文件 |
| `ext:` | `ext:pdf` | 所有PDF文件 |

### Everything工具（进阶推荐）
- 如果Windows自带搜索慢，推荐安装 **Everything**
- 秒搜全盘文件，极为高效
- 官网：https://www.voidtools.com/

## 5️⃣ 综合实战：整理你的电脑

### 实战任务：从零整理一个混乱的电脑

**Step 1：清理桌面**
1. 把桌面上的文件分类归档到对应文件夹
2. 创建文件夹结构：
```
D:\我的文件\
├── 学习\
├── 工作\
├── 娱乐\
└── 临时\
```

**Step 2：清理开始菜单**
1. 右键不需要的程序 → 取消固定
2. 把常用程序固定到开始菜单或任务栏

**Step 3：清理开机启动项**
1. 任务管理器 → 启动
2. 禁用不必要的开机启动程序

**Step 4：清理磁盘**
1. 运行磁盘清理
2. 检查C盘空间，确保至少20%空闲
3. 开启存储感知

**Step 5：安全检查**
1. Windows安全中心 → 快速扫描
2. 检查Windows更新
3. 创建系统还原点

**Step 6：备份重要文件**
1. 把重要文件复制到U盘或云盘
2. 确认文件历史记录是否开启

1. **多用多练**：快捷键开始记不住没关系，用着用着就记住了
