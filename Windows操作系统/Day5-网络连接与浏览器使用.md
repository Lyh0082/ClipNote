# 🌐 Day 5：网络连接与浏览器使用

> **学习目标**：学会连接WiFi和有线网络、掌握浏览器基本操作、了解网络安全常识、学会下载和管理文件。

---

## 1️⃣ 连接网络

### 连接WiFi
1. 点击任务栏右下角的 **网络图标** 🌐
2. 在WiFi列表中找到你的网络名称
3. 点击 **连接**
4. 输入WiFi密码 → 点击"下一步"
5. 显示"已连接"即可

### 连接有线网络
- 将网线插入电脑的网口，Windows会自动连接
- 不需要额外设置，插上就能用

### 网络问题排查
如果连不上网：
1. 确认WiFi是否开启（笔记本可能有物理开关或 `Fn + F键`）
2. 尝试"忘记网络"后重新连接
3. 右键网络图标 → **疑难解答**（Windows自带修复工具）
4. 重启路由器或电脑

## 2️⃣ 浏览器基础（以Edge为例）

Windows 11自带 **Microsoft Edge** 浏览器，已经很好用了。

### 打开浏览器
- 任务栏上的蓝色 Edge 图标点一下
- 或 `Win + S` 搜索"Edge"

### 浏览器界面

```
┌──────────────────────────────────────────────┐
│  ← → 🔄  |  地址栏/搜索框              | ⋯  │
├──────────────────────────────────────────────┤
│                                              │
│              网页内容区                       │
│                                              │
├──────────────────────────────────────────────┤
│                    状态栏（部分网站）          │
└──────────────────────────────────────────────┘
```

### 标签页管理
| 操作 | 方法 |
|------|------|
| 打开新标签页 | 点击标签栏 "+" 或 `Ctrl + T` |
| 关闭当前标签 | 点击标签上的 ✖ 或 `Ctrl + W` |
| 切换标签 | `Ctrl + Tab` 或直接点击标签 |
| 恢复关闭的标签 | `Ctrl + Shift + T` |
| 固定标签 | 右键标签 → 固定标签页 |

## 3️⃣ 浏览器核心技能

### 网址输入
- 在**地址栏**输入网址 → 按 `Enter`
- 网址格式：`https://www.示例.com`
- 不知道网址？直接在地址栏输入关键词搜索

### 搜索技巧
- 直接在地址栏输入关键词即可搜索
- 搜索技巧：
  - 加引号搜精确匹配：`"Windows教程"`
  - 用 `site:` 限定网站：`Windows site:microsoft.com`
  - 用 `filetype:` 搜文件类型：`报告 filetype:pdf`

### 下载文件
1. 在网页上找到下载链接 → 点击
2. 浏览器会弹出下载提示 → 选择 **保存**
3. 下载完成后可在 `C:\Users\你的用户名\Downloads` 找到
4. `Ctrl + J` 打开下载管理查看下载列表

> ⚠️ 下载后记得用杀毒软件扫描一遍！

### 书签/收藏
- `Ctrl + D` 将当前网页加入收藏
- 在收藏夹栏拖拽可以整理书签

## 5️⃣ 网络安全常识

### 🔒 密码安全
- 不要所有网站用同一个密码
- 密码包含大小写字母+数字+符号，长度8位以上
- 建议使用密码管理器（如Edge自带的密码管理）

### 🛡 浏览安全
- 认准网址前的🔒锁标（HTTPS加密）
- 不点击来源不明的链接
- 不下载来路不明的文件
- 警惕弹窗广告（关闭时点 ✖，别点"确定"）
- 不在公共WiFi上登录银行等重要账户

### 隐私保护
- 定期清除浏览数据：`Ctrl + Shift + Delete`
- 选择清除：浏览历史、Cookie、缓存
- 无痕浏览：`Ctrl + Shift + N`（浏览记录不会保存）
