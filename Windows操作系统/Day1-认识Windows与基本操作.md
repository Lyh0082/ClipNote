# 📘 Day 1：认识Windows与基本操作

> **学习目标**：了解Windows是什么、认识桌面环境、掌握鼠标和键盘基本操作、学会开机/关机/重启。

---

## 1️⃣ Windows是什么？

**Windows** 是微软公司开发的操作系统——你可以把它理解为电脑的"大管家"。它负责管理电脑的所有硬件（屏幕、键盘、鼠标、硬盘等）和软件（你安装的各种程序）。

目前主流的Windows版本是 **Windows 10** 和 **Windows 11**（本课程以Windows 11为例，Windows 10操作类似）。

---

## 2️⃣ 认识桌面环境

当你开机进入系统后，看到的第一个画面就是**桌面**。桌面由几个关键部分组成：

### ① 桌面图标
- 桌面上的小图块，双击可以打开对应的程序或文件
- 常见图标：**此电脑**、**回收站**、**控制面板**等

### ② 任务栏（屏幕底部）
任务栏是最常用的操作区域，从左到右依次是：
- **开始按钮** 🪟 — 点击可以打开开始菜单，找到所有已安装的程序
- **搜索框** 🔍 — 输入关键词可以搜索文件、程序或网络内容
- **任务视图** — 查看所有打开的窗口
- **固定/运行中的程序** — 常用程序可以固定在任务栏上
- **系统托盘**（右下角）— 显示时间、音量、网络状态、电池等

### ③ 开始菜单
点击任务栏最左侧的Windows图标即可打开。在这里你可以：
- 浏览所有已安装的程序列表
- 访问"设置""文件资源管理器""电源"等系统功能
- 固定常用程序到开始菜单或任务栏

## 3️⃣ 鼠标的四种基本操作

| 操作 | 方法 | 作用 |
|------|------|------|
| **左键单击** | 按一下左键 | 选中一个图标或按钮 |
| **左键双击** | 快速按两下左键 | 打开文件或程序 |
| **右键单击** | 按一下右键 | 弹出快捷菜单（常用！） |
| **拖拽** | 按住左键不放移动鼠标 | 移动窗口、选中多个文件 |

💡 **小技巧**：遇到不知道怎么办的时候，**右键试试**——Windows会把常用操作放在右键菜单里。

## 4️⃣ 窗口的基本操作

每个打开的程序都会显示在一个**窗口**中。所有窗口都有三个标准按钮（右上角）：

| 按钮 | 图标 | 作用 |
|------|------|------|
| **最小化** | ➖ | 将窗口隐藏到任务栏，程序仍在运行 |
| **最大化/还原** | ⬜ | 让窗口铺满整个屏幕 / 恢复原来大小 |
| **关闭** | ✖ | 关闭程序 |

还可以：
- **移动窗口**：拖拽窗口顶部标题栏d
- **调整窗口大小**：拖拽窗口的四个边或角
- **切换窗口**：按 `Alt + Tab` 快速切换

## 5️⃣ 键盘基础

不需要记住所有键，先认识这**几个最常用**的：

| 键 | 作用 |
|----|------|
| **Enter（回车）** | 确认输入、换行 |
| **Space（空格）** | 输入空格 |
| **Backspace** | 删除光标前面的字符 |
| **Delete** | 删除选中的内容 |
| **Tab** | 跳到下一个输入框 |
| **Caps Lock** | 大写锁定（按一下开/关，注意键盘上的灯） |
| **Shift** | 按住配合字母键输入大写字母 |
| **Ctrl** | 配合其他键实现快捷操作 |
| **Windows键** 🪟 | 打开/关闭开始菜单 |

### 🔥 最实用的快捷键（今天先记3个）

| 快捷键 | 作用 |
|--------|------|
| `Win + D` | 快速回到桌面（再按一次恢复窗口） |
| `Win + E` | 打开文件资源管理器 |
| `Alt + F4` | 关闭当前窗口/关机 |

## 6️⃣ 正确开机与关机

### 开机
按电脑主机或笔记本上的**电源按钮**，等待系统加载完成即可。

### 正确关机（重要！）
**不要直接按电源键强制关机！** 正确步骤：
1. 点击**开始按钮** 🪟
2. 点击**电源图标** ⏻
3. 选择**关机**

> ⚠️ 强制关机可能导致未保存的文件丢失，甚至损坏系统。

### 重启 vs 关机
- **关机**：完全关闭电脑
- **重启**：重新启动系统（安装更新或安装软件后通常需要重启）
