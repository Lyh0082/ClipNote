# ⚙️ Day 3：系统设置与个性化

> **学习目标**：掌握Windows设置的使用方法、学会个性化桌面和任务栏、了解用户账户管理、调整显示和声音设置。

---

## 1️⃣ 打开设置

三种方式：**
1. **快捷键**：`Win + I`（最方便）
2. **开始菜单**：点击开始 🪟 → 点击⚙️ 设置图标
3. **右键开始**：右键点击开始按钮 → 选择"设置"

设置界面分为左右两栏：**左侧是分类菜单**，**右侧是具体设置项**。

---

## 2️⃣ 个性化你的电脑

### 更换壁纸
1. `Win + I` 打开设置 → **个性化** → **背景**
2. 可选：图片 / 幻灯片 / 纯色
3. 点击"浏览照片"选择你喜欢的图片

### 更换主题色
1. 设置 → 个性化 → **颜色**
2. 选择"选择模式"：浅色 / 深色 / 自定义
3. 在"强调色"中选择你喜欢的颜色

### 锁屏界面
设置 → 个性化 → **锁屏界面**
- 可以更换锁屏壁纸
- 可以在锁屏显示日历、天气等小组件

---

## 3️⃣ 任务栏设置

### 固定程序到任务栏
1. 找到程序（开始菜单或桌面）
2. **右键**程序图标 → **固定到任务栏**

### 取消固定
**右键**任务栏上的程序图标 → **从任务栏取消固定**

### 任务栏行为
设置 → 个性化 → **任务栏**：
- 隐藏任务栏（自动隐藏）
- 调整任务栏位置（左/居中）
- 选择显示哪些图标

---

## 4️⃣ 显示设置

### 调整分辨率
1. 设置 → **系统** → **显示器**
2. 在"显示分辨率"下拉菜单选择推荐值（通常标有"推荐"）

### 缩放比例
如果文字太小或太大：
1. 设置 → 系统 → 显示器 → **缩放**
2. 选择 100% / 125% / 150% 等

### 夜间模式
长时间用电脑护眼：
1. 设置 → 系统 → 显示器 → **夜间模式**
2. 开启后屏幕变暖色，减少蓝光

---

## 5️⃣ 声音设置

### 调节音量
- 点击任务栏右下角**音量图标**🔊 → 拖动滑块
- 或使用键盘上的音量键（如有）

### 更换提示音
1. 设置 → **个性化** → **主题** → **声音**
2. 在"程序事件"列表中选择要更改的事件
3. 点击"浏览"选择新的提示音文件

### 选择输出设备
如果你有多个音频设备（耳机、音箱）：
1. 点击音量图标 → 点击音量滑块上方的设备名
2. 选择你想要的输出设备

---

## 6️⃣ 用户账户管理

### 查看当前账户
1. 设置 → **账户** → **你的信息**
2. 可以看到当前登录的账户类型（管理员/标准用户）

### 创建新用户
1. 设置 → 账户 → **其他用户**
2. 点击"将其他人添加到这台电脑"
3. 输入邮箱或创建本地账户

> 💡 建议日常使用标准用户账户，需要安装软件时用管理员权限，更安全。

### 设置/修改密码
1. 设置 → 账户 → **登录选项**
2. 可选：密码 / PIN / 图片密码 / 指纹
3. PIN 比 密码更简短安全，推荐使用

---

## 7️⃣ 通知和专注助手

### 管理通知
1. 设置 → **系统** → **通知**
2. 可以开关各类应用的通知
3. 通知出现的「操作中心」可通过 `Win + A` 打开

### 专注助手
- **关闭**：收到所有通知
- **仅优先级**：只显示重要通知
- **仅闹钟**：屏蔽所有通知（适合专注工作）

快捷操作：点击右下角通知图标 → 点击"专注助手"切换模式
