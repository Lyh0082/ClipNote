# 📁 Day 2：文件管理与资源管理器

> **学习目标**：理解文件和文件夹的概念、熟练使用资源管理器、掌握文件的创建/复制/移动/删除/重命名。

---

## 1️⃣ 什么是文件和文件夹？

电脑里的所有数据都以**文件**的形式存储：

- **文件**：存储信息的基本单位，比如一个文档、一张图片、一首歌
- **文件夹**：用来存放文件的"容器"，类似现实中的文件夹（可以在文件夹里再建文件夹）

文件有不同**类型**，通过**扩展名**区分：

| 扩展名 | 文件类型 | 示例 |
|--------|----------|------|
| `.txt` | 文本文件 | 笔记.txt |
| `.docx` | Word文档 | 报告.docx |
| `.xlsx` | Excel表格 | 成绩单.xlsx |
| `.jpg/.png` | 图片文件 | 风景.jpg |
| `.mp4` | 视频文件 | 录像.mp4 |
| `.mp3` | 音频文件 | 歌曲.mp3 |
| `.pdf` | PDF文档 | 手册.pdf |

💡 **小技巧**：如果看不到扩展名，在资源管理器中点击"查看" → 勾选"文件扩展名"。

---

## 2️⃣ 资源管理器详解

按 `Win + E` 打开**文件资源管理器**，这是你管理文件最重要的工具！

### 界面组成

```
┌─────────────────────────────────────────┐
│  ← → ↑  |  地址栏（显示当前路径）         │
├──────┬──────────────────────────────────┤
│      │                                  │
│ 导航 │        文件列表区                 │
│  窗格 │                                  │
│      │                                  │
│ 快速 ├──────────────────────────────────┤
│ 访问 │  详情窗格（选中文件后显示信息）      │
└──────┴──────────────────────────────────┘
```

### 导航窗格（左侧）

- **快速访问**：固定常用文件夹，一键到达
- **此电脑**：查看所有磁盘驱动器（C盘、D盘等）
- **网络**：查看局域网共享资源

### 关键概念：路径

文件在电脑中的位置用**路径**表示：
- `C:\Users\你的用户名\Documents\报告.docx`
- 路径就像文件的"家庭地址"

## 3️⃣ 文件基本操作

### 创建文件夹
1. 在资源管理器中打开你想创建的位置
2. **右键空白处** → **新建** → **文件夹**
3. 输入文件夹名称 → 按 `Enter`

### 创建文件
1. **右键空白处** → **新建** → 选择文件类型（如文本文档）
2. 输入文件名 → 按 `Enter`

### 选中文件
| 操作 | 方法 |
|------|------|
| 选中一个 | 左键单击 |
| 选中多个（连续） | 单击第一个 → 按住`Shift` → 单击最后一个 |
| 选中多个（不连续） | 按住`Ctrl` → 逐个单击 |
| 全选 | `Ctrl + A` |

### 复制文件
方法一：**右键** → **复制** → 到目标位置 **右键** → **粘贴**
方法二：选中 → `Ctrl + C` → 到目标位置 → `Ctrl + V`

### 移动文件
方法一：**右键** → **剪切** → 到目标位置 **右键** → **粘贴**
方法二：选中 → `Ctrl + X` → 到目标位置 → `Ctrl + V`
方法三：直接**拖拽**文件到目标文件夹

> 复制和移动的区别：复制 = 原文件保留一份副本；移动 = 原文件转移到新位置。

### 重命名文件
方法一：**右键** → **重命名** → 输入新名 → `Enter`
方法二：选中文件 → 按 `F2` → 输入新名 → `Enter`

### 删除文件
方法一：**右键** → **删除**（文件进入回收站）
方法二：选中 → 按 `Delete` 键
方法三：永久删除 → `Shift + Delete`（⚠️ 不经过回收站，无法恢复！）

---

## 4️⃣ 回收站——误删文件的"后悔药"

删除的文件会先进入**回收站**，可以恢复：
1. 双击桌面上的**回收站**图标
2. 找到要恢复的文件 → **右键** → **还原**
3. 文件会回到删除前的位置

> ⚠️ 用 `Shift + Delete` 删除的文件不进回收站，无法通过这种方式恢复。

### 清空回收站
右键回收站 → **清空回收站** → 确认（⚠️ 不可恢复！）

---

## 5️⃣ 搜索文件

在资源管理器的**搜索框**中输入关键词：
- 按文件名搜索：直接输入文件名的一部分
- 按类型搜索：输入 `*.jpg` 搜索所有图片
- 按日期搜索：点击搜索框 → 出现"修改日期"筛选

快捷键：`Win + S` 打开全局搜索

---

## 6️⃣ 文件管理的好习惯

1. **分类存放**：按用途建文件夹，如"学习""工作""娱乐"
2. **命名规范**：文件名要有意义，如"2026-05-06_会议笔记.docx"
3. **别放桌面**：桌面文件占C盘空间，建议存在D盘或文档中
4. **定期整理**：每周花5分钟整理文件，避免桌面混乱
5. **重要文件备份**：复制到U盘或云盘，以防万一
