# 🛡️ Day 6：系统安全与维护

> **学习目标**：了解Windows安全防护、学会系统更新、掌握磁盘清理和优化方法、认识常见电脑问题及解决办法。

---

## 1️⃣ Windows安全中心

按 `Win + I` → **隐私和安全性** → **Windows 安全中心**

### 核心防护模块

| 模块 | 作用 | 建议 |
|------|------|------|
| **病毒和威胁防护** | 查杀病毒、实时保护 | 保持开启 |
| **防火墙和网络保护** | 阻止恶意网络访问 | 保持开启 |
| **应用和浏览器控制** | 过滤恶意网站和下载 | 保持开启 |
| **设备安全性** | 核心隔离等硬件级保护 | 默认即可 |

### 手动扫描电脑
1. 打开 Windows 安全中心
2. 点击 **病毒和威胁防护**
3. 点击 **扫描选项** → 选择：
   - **快速扫描**（几分钟，检查常见位置）
   - **完全扫描**（较慢，检查所有文件）
   - **自定义扫描**（指定文件夹）

> 💡 不需要额外安装杀毒软件！Windows Defender 已经足够好用。

## 2️⃣ Windows更新

### 为什么需要更新？
- 修复安全漏洞（最重要！）
- 获得新功能
- 修复已知bug
- 提升系统稳定性

### 检查更新
1. `Win + I` → **Windows 更新**
2. 点击 **检查更新**
3. 如果有可用更新 → 点击 **下载并安装**
4. 安装完成后可能需要**重启电脑**

### 设置自动更新
1. `Win + I` → Windows 更新 → **高级选项**
2. 确保以下选项开启：
   - 接收其他Microsoft产品的更新
   - 下载更新后自动重启

> ⚠️ 不要关闭Windows更新！很多人为了省事关闭更新，这非常危险。

## 3️⃣ 磁盘清理

### 方法一：系统自带磁盘清理
1. `Win + S` 搜索"磁盘清理"
2. 选择要清理的盘符（通常是C盘）
3. 勾选要清理的文件类型
4. 点击 **确定** → **删除文件**

可安全清理的项目：
- ✅ 临时文件
- ✅ 回收站
- ✅ 下载的程序文件
- ✅ 临时Internet文件
- ❌ 不要清理系统文件（除非你确定知道在做什么）

### 方法二：存储感知（推荐）
1. `Win + I` → **系统** → **存储**
2. 开启 **存储感知**（自动清理临时文件）
3. 可以设置自动清理规则（如30天自动清理回收站）

### 查看磁盘空间
1. `Win + I` → 系统 → 存储
2. 可以看到各类文件占用的空间
3. 点击各分类可以进一步清理

## 4️⃣ 任务管理器

### 打开方式
- `Ctrl + Shift + Esc`（最快）
- `Ctrl + Alt + Delete` → 任务管理器
- 右键任务栏空白处 → 任务管理器

### 用途

#### 查看程序状态
- **进程**标签：查看所有正在运行的程序
- 可以看到CPU、内存、磁盘、网络的使用情况

#### 强制关闭卡住的程序
1. 找到卡住的程序
2. 右键 → **结束任务**

#### 查看开机启动项
1. 点击 **启动应用** 标签
2. 禁用不需要的开机启动项 → 加快开机速度
3. 右键不需要的程序 → **禁用**

> ⚠️ 不要结束名称为"Windows"开头或"System"的系统进程！

## 5️⃣ 常见问题与解决

### 电脑变慢了
| 原因 | 解决方法 |
|------|----------|
| 开机启动项太多 | 任务管理器 → 启动 → 禁用不必要的 |
| 磁盘空间不足 | 清理C盘，保留至少20%空间 |
| 杀毒软件冲突 | 只保留Windows Defender |
| 临时文件太多 | 运行磁盘清理或开启存储感知 |
| 电脑过热 | 清理散热口灰尘，放硬质表面 |

### 程序无响应
1. 按一次 `Esc` 等几秒
2. 按 `Alt + F4` 尝试关闭
3. `Ctrl + Shift + Esc` 打开任务管理器 → 结束任务
4. 最后一招：长按电源键强制关机（仅在以上方法无效时使用）

### 蓝屏问题
- 记下错误代码
- 重启电脑
- 如果反复出现 → 更新驱动或搜索错误代码
- 进阶：使用事件查看器查看错误日志

## 6️⃣ 数据备份

### 为什么要备份？
硬盘可能损坏、电脑可能中毒、文件可能误删——备份是最后一道防线。

### 备份方法

**方法一：文件历史记录**
1. 准备一个外接硬盘或大容量U盘
2. `Win + I` → **系统** → **存储** → **高级存储设置** → **备份选项**
3. 开启 **自动备份我的文件**

**方法二：手动备份**
- 定期把重要文件复制到U盘或移动硬盘
- 或上传到云盘（OneDrive、百度网盘等）

**方法三：系统还原点**
1. `Win + S` 搜索"创建还原点"
2. 点击 **创建** → 输入名称 → 确认
3. 出问题后可以回滚到这个时间点
