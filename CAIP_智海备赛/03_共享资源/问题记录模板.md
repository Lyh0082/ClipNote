## 常用调试技巧

### 1. 检查Tensor形状
```python
def debug_tensor(x, name="tensor"):
    """打印Tensor信息"""
    print(f"[{name}] shape={x.shape}, dtype={x.dtype}, "
          f"device={x.device}, range=[{x.min():.3f}, {x.max():.3f}]")
```

### 2. 检查数据加载
```python
# 检查DataLoader是否正常工作
images, labels = next(iter(train_loader))
print(f"Batch images shape: {images.shape}")  # 应该是 [B, C, H, W]
print(f"Batch labels shape: {labels.shape}")  # 应该是 [B]
print(f"Label range: {labels.min()} ~ {labels.max()}")  # 应该在0~num_classes-1
```

### 3. 检查模型输出
```python
# 检查模型输出
dummy_input = torch.randn(4, 3, 224, 224)
output = model(dummy_input)
print(f"Model output shape: {output.shape}")  # 应该是 [4, num_classes]
print(f"Output sample: {output[0]}")  # 检查数值范围
```

### 4. 常见错误及解决

```text
❌ "CUDA out of memory"
   → 减小batch_size
   → 使用梯度累积
   → 检查是否有显存泄漏

❌ "Expected tensor to be on GPU, but found on CPU"
   → 确保model和input都在同一个device
   → model.to(device), input.to(device)

❌ "shape mismatch"
   → 检查模型输入输出通道数
   → 检查全连接层输入维度

❌ "No module named 'xxx'"
   → pip install xxx

❌ "FileNotFoundError"
   → 检查路径是否正确
   → 使用os.path.join()拼接路径
   → 使用绝对路径测试
```

### 5. PyTorch调试建议
```python
# 开启异常检测（定位梯度问题）
torch.autograd.set_detect_anomaly(True)

# 监控梯度范数
total_norm = 0
for p in model.parameters():
    if p.grad is not None:
        param_norm = p.grad.data.norm(2)
        total_norm += param_norm.item() ** 2
total_norm = total_norm ** 0.5
print(f"Gradient norm: {total_norm:.4f}")
# 梯度太大 → 学习率太大/Loss爆炸
# 梯度太小/为零 → 梯度消失/模型没有学习
```
