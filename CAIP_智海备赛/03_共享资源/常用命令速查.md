# 常用命令速查

## 环境管理
```bash
# Conda
conda create -n caip python=3.9      # 创建环境
conda activate caip                    # 激活
conda deactivate                       # 退出
conda env list                         # 查看环境列表
conda remove -n caip --all             # 删除环境

# pip
pip install package_name               # 安装包
pip install -r requirements.txt        # 批量安装
pip freeze > requirements.txt          # 导出依赖
pip list                               # 查看已安装
```

## PyTorch常用操作速查

### Tensor操作
```python
torch.randn(3, 224, 224)       # 随机初始化
torch.zeros(3, 224)            # 全零
torch.ones(3, 224)             # 全一
torch.tensor([1, 2, 3])        # 从列表创建
x.shape                        # 查看形状
x.device                       # 查看设备
x.dtype                        # 查看数据类型
x.to(device)                   # 切换到设备
x.cpu()                        # 转到CPU
x.numpy()                      # 转NumPy
torch.from_numpy(arr)          # NumPy转Tensor
x.unsqueeze(0)                 # 加维度
x.squeeze(0)                   # 减维度
x.permute(1, 2, 0)             # 重排维度
```

### 模型操作
```python
model = ModelClass()           # 创建模型
model.to(device)               # 移到设备
model.train()                  # 训练模式
model.eval()                   # 评估模式
model.parameters()             # 获取参数

torch.save(model.state_dict(), 'model.pth')           # 保存模型
model.load_state_dict(torch.load('model.pth'))        # 加载模型

# 完整保存
torch.save({
    'epoch': epoch,
    'model_state_dict': model.state_dict(),
    'optimizer_state_dict': optimizer.state_dict(),
    'best_f1': best_f1,
}, 'checkpoint.pth')
```

### 训练循环
```python
optimizer.zero_grad()         # 清零梯度
loss.backward()               # 反向传播
optimizer.step()              # 更新参数
with torch.no_grad():         # 禁用梯度
    ...
```

## 评估指标
```python
from sklearn.metrics import f1_score, confusion_matrix, classification_report

f1_score(y_true, y_pred, average='macro')          # Macro F1
f1_score(y_true, y_pred, average='weighted')       # Weighted F1
confusion_matrix(y_true, y_pred)                   # 混淆矩阵
classification_report(y_true, y_pred)              # 完整报告
```

## 常用超参数范围
```text
学习率 (lr):      1e-4 ~ 1e-2 (Adam), 1e-3 ~ 1e-1 (SGD)
Batch Size:       16, 32, 64 (视GPU内存而定)
权重衰减 (wd):    1e-5 ~ 1e-3
Dropout:          0.2 ~ 0.5
优化器:           Adam (默认), AdamW (推荐), SGD+Momentum
Epochs:           30 ~ 100 (根据Early Stopping决定)
温度 (T):         2 ~ 6 (知识蒸馏)
剪枝比例:         10% ~ 50%
```
