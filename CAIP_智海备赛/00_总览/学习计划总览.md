# 学习计划总览 — 42天零基础备赛

## 六步框架贯穿全局

> 本备赛计划每个 Cycle 都遵循**六步学习法**，确保学得深、记得住、用得出：

| 步骤 | 核心问题 | 产出 |
|------|----------|------|
| **1. 建立地图** | 这个领域长什么样？我在哪？ | 知识全景图 + 定位 |
| **2. 建立统一抽象** | 表面不同的东西，底层是什么？ | 统一概念表 |
| **3. 核心矛盾** | 最关键的 trade-off 是什么？ | 矛盾对 + 竞赛影响 |
| **4. 因果链/易混点/重点** | 什么导致什么？什么容易搞混？ | 因果链 + 易混表 + 重点标注 |
| **5. 压缩** | 能不能三句话讲完？ | 重建架构 + 本质提炼 + 易错速记 |
| **6. 检索能力检查** | 闭卷能复述/画图/推导吗？ | 复述题 + 画图题 + 代码题 |

## 时间安排说明

- **周期**：5天学习 → 1天测试+复习 → 1天休息
- **总周期**：6个完整周期（42天）≈ 1.5个月
- **时间线**：5月中旬 → 6月底（赛前准备完成）
- **每天计划学习时间**：主输出 3-5小时 / 辅助 2-3小时
