## 二、统一抽象

> **MATLAB的核心思想：一切皆矩阵。**
> 标量是1×1矩阵，向量是1×n或n×1矩阵，图像是矩阵。

## 三、核心矛盾

- **矛盾**：数学公式是连续的，计算机只能算离散的。
- **解决**：数值方法（差分、插值、ODE求解器）把连续问题离散化。

## 四、因果链/易混点/重点

### 因果链：从公式到代码

```
物理定律（微分方程）
  ↓ 离散化
数值方法（差分/Runge-Kutta）
  ↓ 编码
MATLAB函数调用（ode45 / fsolve）
  ↓ 输出
数值解 + 可视化
```

### 语法速查

```matlab
%% 1. 变量与矩阵
a = 3;                     % 标量
v = [1, 2, 3];             % 行向量
M = [1 2; 3 4];            % 2×2矩阵
x = linspace(0, 2*pi, 100); % 0到2π等分100点

%% 2. 运算
A * B      % 矩阵乘法
A .* B     % 逐元素乘法（注意那个点！）
A'         % 转置
inv(A)     % 求逆
det(A)     % 行列式

%% 3. 绘图
plot(x, y, 'r-', 'LineWidth', 2);
xlabel('x'); ylabel('y'); title('标题');
legend('曲线1');
subplot(2,1,1); % 上下两图，选上半部分

%% 4. 解方程
% 符号解
syms x;
sol = solve(x^2 - 3*x + 2 == 0, x);

% 线性方程组 Ax = b
x = A \ b;   % 左除，比inv(A)*b更好

% 非线性方程组
fsolve(@myfun, x0);

%% 5. 常微分方程（A类题核心！）
% dy/dt = f(t, y)，写函数文件或匿名函数
% 例：dy/dt = -2y
f = @(t, y) -2*y;
[t, y] = ode45(f, [0 5], 1);  % t∈[0,5], y(0)=1
plot(t, y);

%% 6. 数据拟合
p = polyfit(x, y, 2);   % 二次多项式拟合
y_fit = polyval(p, x);  % 计算拟合值

%% 7. 条件与循环
for i = 1:10
    if mod(i,2) == 0
        disp([num2str(i), ' 是偶数']);
    end
end
```

### 易混点

| MATLAB写法 | 易错 | 正确理解 |
|-----------|------|---------|
| `A*B` vs `A.*B` | 忘记点 | `*`是矩阵乘，`.*`是逐元素乘 |
| `=` vs `==` | 混淆 | `=`赋值，`==`判断相等 |
| `A\b` vs `inv(A)*b` | 都能解方程 | `A\b`更稳定更快 |
| 数组索引从1开始 | 习惯0开始 | MATLAB索引从1开始！ |

## 第2天：力与受力分析

### 一、建立地图

```
理论力学
├── 静力学 ──← 今天在这里：受力分析是所有力学问题的起点
├── 运动学
└── 动力学
```

### 二、统一抽象

| 现实 | 抽象 | 条件 |
|------|------|------|
| 有大小的物体 | 质点（无大小有质量） | 尺寸<<运动范围 |
| 不变形的物体 | 刚体 | 变形可忽略 |
| 各种具体的力 | 力矢量（大小+方向+作用点） | 统一矢量运算 |

**受力分析的统一流程**：
```
确定研究对象 → 隔离体 → 画主动力 → 画约束反力 → 检查
```

### 三、核心矛盾

> **主动力已知 vs 约束反力未知**
> - 主动力：重力、推力、拉力——题目直接给出
> - 约束反力：支撑力、摩擦力、铰链力——由平衡条件反推

### 四、因果链/易混点

**因果链**：
```
受力分析 → 列平衡方程 → 解未知力
    ↑              ↓
 隔离体画法    ΣFx=0, ΣFy=0, ΣM=0
```

**易混点**：

| 概念 | 易混 | 正解 |
|------|------|------|
| 作用力反作用力 vs 二力平衡 | 都等大反向 | 作用力反作用力作用于**两个物体**；二力平衡作用于**同一物体** |
| 铰链约束 vs 固定端约束 | 约束反力不同 | 铰链：2个力分量；固定端：2个力分量+1个力矩 |
| 摩擦力方向 | 主观判断 | 摩擦力方向与**相对运动趋势**相反 |

### 五、压缩提炼

> **受力分析的本质：把一个「被约束的物体」从环境中「隔离」出来，用「力」替代所有外界影响。**

**常见约束的约束反力一览**：
```
光滑面接触 → 1个力（法向）
粗糙面接触 → 2个力（法向+摩擦）
铰链       → 2个力（水平+竖直）
固定端     → 2个力+1个力矩
柔索       → 1个拉力（沿绳方向）
```

1. 画出「粗糙斜面上静止的木块」受力图，写出平衡方程。

## 第3天：力矩与力偶

### 一、建立地图

```
静力学
├── 力的合成（第2天：汇交力系）
├── 力矩与力偶 ← 今天
└── 一般力系简化
```

### 二、统一抽象

```
力矩 = 力 × 力臂 = r × F （矢量叉乘）
  → 描述力使物体绕某点转动的效果

力偶 = 大小相等、方向相反、不共线的一对力
  → 纯转动效应，与参考点无关
  → 力偶矩 M = F × d（力×力偶臂）
```

**核心抽象**：力对物体的外效应 = 移动（合力）+ 转动（合力矩/力偶矩）

### 三、核心矛盾

> **力矩依赖参考点，力偶不依赖参考点。**
> 这是力偶存在的意义——它代表一种「纯粹的转动」，不受观察点选择的影响。

### 四、因果链/易混点

**因果链**：
```
力的平移定理：
  作用在A点的力F，等效于
  → 作用在O点的力F + 力矩M= r×F
  （力平移，必须补一个力矩）
```

**易混点**：

| 概念 | 区别 |
|------|------|
| 力矩 M₀(F) = r×F | 依赖于参考点O |
| 力偶矩 M = F×d | 与参考点无关 |
| 合力矩定理 | 合力对某点的矩 = 各分力对该点矩的代数和 |

### 五、压缩提炼

> **力矩和力偶的关系**：力偶是力矩的「可携带版本」——无论你站在哪里观察，力偶矩不变。力的平移产生了力偶，力偶是平移的「代价」。

### 六、检索能力检查 + MATLAB

1. 解释：为什么力偶矩与参考点无关？
2. 推导：力F=(3,4)N作用在点A=(1,2)m，求对原点O的力矩。
3. MATLAB：编写程序，给定若干力和作用点，计算合力与合力矩。

```matlab
%% 第3天 MATLAB：力系简化
% 输入：若干力 [Fx, Fy] 和作用点 [x, y]
forces = [3 4; -2 5; 1 -3];    % 各力的分量
points = [1 2; 3 0; -1 1];     % 各力作用点

% 合力
F_total = sum(forces, 1);

% 对原点的合力矩（2D: M = x*Fy - y*Fx）
M_total = 0;
for i = 1:size(forces,1)
    M_total = M_total + points(i,1)*forces(i,2) - points(i,2)*forces(i,1);
end

fprintf('合力 = [%.2f, %.2f] N\n', F_total(1), F_total(2));
fprintf('合力矩 = %.2f N·m\n', M_total);
```

### 一、建立地图

├── 力矩力偶（第3天）
└── 力系简化与平衡 ← 今天：静力学收官

### 二、统一抽象

**力系简化的统一结论**：

```
任意力系 → 简化 → 一个力 + 一个力偶
              (合力)   (合力矩)

特殊情况：
  合力=0, 合力矩=0 → 平衡（静力学核心）
  合力=0, 合力矩≠0 → 力偶（纯转动）

```

### 三、核心矛盾

> **静定 vs 超静定**
> - 未知力数 = 独立平衡方程数 → 静定（可解）
> - 未知力数 > 独立平衡方程数 → 超静定（需材料力学补充变形条件）
> - **这是理论力学与材料力学的分界线！**

平面力系：最多3个独立方程（ΣFx=0, ΣFy=0, ΣM=0）

### 四、因果链/易混点

**因果链**：
```
选研究对象 → 画受力图 → 选坐标/矩心
  → 列方程（3个） → 解未知力（3个）→ 校核
```

**易混点**：
- 二力构件：只受两个力，这两个力必等大反向共线——**先识别二力构件可简化计算**
- 三力平衡汇交定理：三力平衡且不平行 → 必汇交于一点

### 五、压缩提炼

> **静力学的「三步法」：隔离 → 列方程 → 解方程。**
> 超出3个未知力？你需要材料力学（考虑变形协调）。

### 六、检索能力检查 + MATLAB

1. 什么是静定和超静定？举例说明。

## 第5天：摩擦

### 一、建立地图

```
静力学特殊问题
└── 摩擦 ← 今天
    ├── 滑动摩擦
    └── 滚动摩擦（了解）
```

### 二、统一抽象

```
摩擦力的统一描述：
  |F_f| ≤ μ * F_N    （库仑摩擦定律）
  
三种状态：
  未滑动：F_f = 需要的值（≤ μs * F_N）
  临界滑动：F_f = μs * F_N    （最大静摩擦）
  已滑动：F_f = μk * F_N      （动摩擦，μk < μs）
```

### 三、核心矛盾

> **摩擦力的「被动性」**：摩擦力是被动力，大小和方向都由其他力决定，不能独立指定。
> 只有在临界状态下，才能用 F=μN 计算。

### 四、因果链/易混点

**因果链**：
```
外力作用 → 物体有运动趋势 → 摩擦力产生（对抗趋势）
  → 外力增大 → 摩擦力增大（到临界值）
  → 超过临界 → 开始滑动 → 摩擦力降为动摩擦力
```

**易混点**：
- 摩擦力方向：与**相对运动趋势**相反（不是与运动方向相反！静摩擦时物体没动）
- 摩擦角：φ = arctan(μ)，全反力与法线的夹角 ≤ 摩擦角则自锁

### 五、压缩提炼

> **摩擦问题的关键：先判断状态（动/静/临界），再选公式。**
> A类题中摩擦常出现在：斜面问题、带传动、螺栓连接。

### 六、检索能力检查 + MATLAB

1. 什么叫自锁？用摩擦角解释。
2. 一个物体放在倾角30°的斜面上，μ=0.4，会滑动吗？
3. MATLAB：画摩擦力随斜面角度变化图。

```matlab

## 第6天：运动学基础——点的运动

### 一、建立地图

```
理论力学
├── 静力学（第2-5天）✓
├── 运动学 ← 今天开始
│   ├── 点的运动
│   ├── 刚体的基本运动
│   └── 点的合成运动
└── 动力学
```

### 二、统一抽象

**描述运动的三种方法**（殊途同归）：

| 方法 | 位置 | 速度 | 加速度 |
|------|------|------|--------|
| 矢量法 | **r**(t) | **v** = d**r**/dt | **a** = d**v**/dt |
| 直角坐标 | x(t), y(t) | vₓ=ẋ, vᵧ=ẏ | aₓ=ẍ, aᵧ=ÿ |
| 自然坐标 | s(t) | v=ṡ | aₜ=̈s, aₙ=v²/ρ |

**统一本质**：速度是位置的变化率，加速度是速度的变化率。

### 三、核心矛盾

> **自然坐标中的法向加速度 aₙ=v²/ρ**
> - 直线运动：ρ→∞，aₙ=0，只有切向加速度
> - 匀速圆周：aₜ=0，aₙ=v²/R
> - **速度方向改变需要加速度**——这是很多人直觉上容易忽略的

### 四、因果链/易混点

**因果链（A类题核心！）**：
```
位置 r(t) → 微分 → 速度 v(t) → 微分 → 加速度 a(t)
加速度 a(t) → 积分 → 速度 v(t) → 积分 → 位置 r(t)
```

**易混点**：
- 速度大 ≠ 加速度大（匀速圆周运动速度大小不变但有加速度）
- 加速度为零 → 速度恒定（不是速度为零！）

### 五、压缩提炼

> **运动学的本质就是微积分：位置↔速度↔加速度，微分和积分互为逆运算。**

### 六、检索能力检查 + MATLAB

1. 写出匀变速直线运动三大公式（从加速度积分推导）。
2. 匀速圆周运动的加速度方向指向哪里？大小是多少？
3. MATLAB：模拟抛体运动（数模A类经典题型）。

```matlab
%% 第6天 MATLAB：抛体运动（A类题高频）
g = 9.81; v0 = 20; theta0 = deg2rad(45);
vx0 = v0*cos(theta0); vy0 = v0*sin(theta0);

% 解析解
t_end = 2*vy0/g;
t = linspace(0, t_end, 200);
x = vx0 * t;
y = vy0 * t - 0.5*g*t.^2;

figure; plot(x, y, 'r-', 'LineWidth', 2);
xlabel('x (m)'); ylabel('y (m)'); title('抛体运动轨迹');
axis equal; grid on;

% 用ode45数值求解（为后续复杂问题打基础）
% dx/dt = vx, dvx/dt = 0
% dy/dt = vy, dvy/dt = -g

### 二、统一抽象

**平动与转动的类比**（关键抽象！）：

| 概念 | 平动（线量） | 转动（角量） |
|------|------------|------------|
| 位置 | s | θ |
| 速度 | v = ds/dt | ω = dθ/dt |
| 加速度 | a = dv/dt | α = dω/dt |
| 运动方程 | F=ma | M=Jα |
| 动能 | ½mv² | ½Jω² |
| 动量 | mv | Jω |

**线量与角量的桥梁**：v = ωr, a = αr（r为到转轴距离）

### 三、核心矛盾

> **刚体上不同点的运动不同，但角量（ω, α）相同！**
> 这是「刚体」条件的体现：物体不变形 → 各点角量相同。

### 四、因果链/易混点

**因果链**：
```
刚体定轴转动
  → 角运动：θ(t) → ω=dθ/dt → α=dω/dt
  → 线运动：v=ωr, aₜ=αr, aₙ=ω²r
  → 各点：r不同 → 线量不同，但角量相同
```

**易混点**：
- 刚体平动≠直线运动（平动可以是曲线平动，如荡秋千的横梁）
- 定轴转动中，加速度有切向和法向两个分量，不要漏法向

### 五、压缩提炼

> **平动看质心，转动看角量。** 线量与角量通过 v=ωr 桥接。

## 第9天：动力学基础——牛顿定律与动量定理

### 一、建立地图

```
理论力学
├── 静力学 ✓
├── 运动学 ✓
└── 动力学 ← 今天开始
    ├── 牛顿第二定律 F=ma
    ├── 动量定理
    ├── 动量矩定理
    └── 动能定理
```

### 二、统一抽象

**动力学三大定理的统一视角**：

| 定理 | 核心方程 | 本质 | 守恒条件 |
|------|---------|------|---------|
| 动量定理 | d**p**/dt = **F** | 力在**时间**上的累积 | ΣF_ext=0 → p守恒 |
| 动量矩定理 | d**L**/dt = **M** | 力矩在**时间**上的累积 | ΣM_ext=0 → L守恒 |
| 动能定理 | dT = δW | 力在**空间**上的累积 | 保守系统 → T+V守恒 |

**它们都来自牛顿第二定律**，只是做了不同的数学变换：
```
F = ma
  → 两边乘dt → F dt = dp   （动量定理）
  → 两边叉乘r → r×F dt = dL （动量矩定理）
  → 两边点乘dr → F·dr = dT  （动能定理）
```

### 三、核心矛盾

> **内力不改变系统的总动量，但可以改变系统内部的能量分配。**
> - 炸弹爆炸：内力很大，但总动量守恒（系统动量不变）
> - 但动能大幅增加（化学能→动能）

### 四、因果链/易混点

**因果链**：
```
外力 → 冲量 I = ∫F dt → 动量变化 Δp = I
  ↓ 特例
ΣF = 0 → p守恒 → 一维：m₁v₁ + m₂v₂ = 常数
```

**易混点**：
- 动量守恒 ≠ 机械能守恒（完全非弹性碰撞：动量守恒，动能不守恒）
- 内力冲量等大反向，不改变总动量

### 五、压缩提炼

> **三大定理同一源：牛顿第二定律。** 选哪个定理？看题目问什么：
> - 求速度/力与时间关系 → 动量定理
> - 求角速度/力矩 → 动量矩定理
> - 求速度/做功 → 动能定理

## 第10天：振动基础——A类题高频考点

### 一、建立地图

```
动力学专题
└── 振动 ← 今天（A类题出题频率极高）
    ├── 自由振动（无阻尼）
    ├── 阻尼振动
    └── 受迫振动（外激励）
```

### 二、统一抽象

**振动问题的统一模型**：

```
质量-弹簧-阻尼系统 (m-c-k)
    mẍ + cẋ + kx = F(t)

三种情况：
  自由振动：   mẍ + kx = 0        → x = A sin(ωₙt + φ)
  阻尼振动：   mẍ + cẋ + kx = 0   → 衰减振荡
  受迫振动：   mẍ + cẋ + kx = F₀sin(ωt) → 稳态+瞬态
```

**关键参数**：
```
ωₙ = √(k/m)      固有频率
ζ = c/(2√(mk))    阻尼比
ωd = ωₙ√(1-ζ²)    阻尼固有频率
```

### 三、核心矛盾

> **共振：当外力频率 ω ≈ ωₙ 时，振幅急剧增大！**
> - 无阻尼共振 → 振幅→∞（理论）
> - 有阻尼共振 → 振幅有上限，但仍很大
> - 工程意义：避开共振或增加阻尼

### 四、因果链/易混点

**因果链**：
```
振动微分方程
  ↓ 特征方程
判别式 Δ = (c/m)² - 4k/m
  ├── Δ < 0: 欠阻尼 → 衰减振荡（工程常见）
  ├── Δ = 0: 临界阻尼 → 最快回到平衡
  └── Δ > 0: 过阻尼 → 缓慢回到平衡（不振荡）
```

**易混点**：
- 固有频率只与m和k有关，与初始条件无关
- 阻尼振动的「频率」是ωd，不是ωₙ
- 受迫振动的稳态频率 = 激励频率ω，不是固有频率ωₙ

### 五、压缩提炼

> **振动的本质：质量（惯性）和弹簧（恢复力）的博弈。** 阻尼消耗能量，外力注入能量。
> A类题中，振动问题几乎100%需要用ode45数值求解。

### 六、检索能力检查 + MATLAB

## 第12天：材料力学地图 + 内力与应力

### 一、建立地图

├── 强度理论：什么条件下材料破坏？
└── 稳定性：压杆屈曲
```

### 二、统一抽象

**从理论力学到材料力学的关键转变**：

```
理论力学：刚体（不变形）→ 只关心运动
材料力学：变形体 → 关心内部怎么受力、怎么变形

核心抽象：
  外力 → 内力（截面法）→ 应力（单位面积内力）→ 变形/破坏
```

**截面法**（材料力学最基本的方法）：
```
1. 假想切开
2. 取出一部分
3. 用内力代替另一部分的作用
4. 列平衡方程求内力
```

**应力 = 内力/面积**：
```
正应力 σ = N/A  （垂直于截面，拉为正）
切应力 τ = V/A  （平行于截面）
```

### 三、核心矛盾

> **强度 vs 刚度 vs 稳定性**
> - 强度：会不会断？（应力 ≤ 许用应力）
> - 刚度：变形大不大？（变形 ≤ 许用变形）
> - 稳定性：会不会突然失稳？（压力 ≤ 临界力）
> - 三个要求，一个不满足就设计失败

### 四、因果链/易混点

**因果链**：
```
外力 → 内力（截面法）→ 应力（除以面积）→ 与许用应力比较
  → 强度条件：σ ≤ [σ]
```

**易混点**：
- 内力是「截面上力的合力」，不是某个点的力
- 应力是「单位面积上的内力」，是内力的集度
- 正应力σ和切应力τ是应力的两个分量，不同方向

### 五、压缩提炼

> **材料力学的思维模式：外力→内力→应力→强度校核。**
> 截面法是起点，强度条件是终点。

  外力沿轴线 → 轴力N → 正应力σ = N/A → 纵向应变ε = σ/E
  
胡克定律：σ = Eε
  E：弹性模量（材料刚度，钢≈200GPa，铝≈70GPa）

变形：ΔL = NL/(EA)
  EA：抗拉刚度（越大越难拉长）
```

### 三、核心矛盾

> **强度要求（A要大）vs 经济要求（A要小）**
> 设计截面面积A：σ = N/A ≤ [σ] → A ≥ N/[σ]
> 安全与经济的平衡，A类优化题的核心。

### 四、因果链/易混点

**因果链**：
```
外力F → 轴力N = F → 正应力σ = N/A
  → 应变ε = σ/E → 变形ΔL = εL = NL/(EA)
  → 强度条件：σ ≤ [σ] → A ≥ N/[σ]
```

**易混点**：
- 弹性模量E是材料性质，与截面尺寸无关
- 刚度EA = 弹性模量×截面积，与两者都有关
- 泊松比μ：横向应变 = -μ × 纵向应变（拉长时变细）

### 五、压缩提炼

> **拉压问题一句话：力÷面积=应力，应力÷弹性模量=应变，应变×长度=变形。**

### 六、检索能力检查 + MATLAB

1. 推导 ΔL = NL/(EA)。
2. 什么是抗拉刚度？单位是什么？
3. MATLAB：阶梯杆的变形计算。

## 第15天：扭转

### 一、建立地图

```
四种基本变形
└── 扭转 ← 今天
    → 传动轴、钻杆
```

### 二、统一抽象

```
扭转的统一模型：
  外力偶 → 扭矩T → 切应力τ = Tρ/Iₚ → 最大切应力τmax = T/Wₚ
  
  Iₚ：极惯性矩（截面抗扭几何量）
    实心圆轴：Iₚ = πd⁴/32
    空心圆轴：Iₚ = π(D⁴-d⁴)/32
  Wₚ：抗扭截面系数 = Iₚ/R
  
扭转角：φ = TL/(GIₚ)
  GIₚ：抗扭刚度
```

### 三、核心矛盾

> **实心轴 vs 空心轴：为什么传动轴常做成空心？**
> - 扭转切应力与半径成正比：中心应力小，边缘大
> - 实心轴中心材料「浪费」→ 空心轴更高效
> - 同样重量下，空心轴的抗扭能力更强

### 四、因果链/易混点

**因果链**：
```
外力偶 → 扭矩T → 切应力分布τ(ρ) = Tρ/Iₚ
  → 强度条件：τmax = T/Wₚ ≤ [τ]
  → 刚度条件：φ/L = T/(GIₚ) ≤ [θ]（单位长度扭转角）
```

**易混点**：
- 极惯性矩Iₚ vs 惯性矩I（弯曲用）——下节讲
- 扭转切应力沿径向线性分布，弯曲正应力沿高度线性分布

### 五、压缩提炼

> **扭转 vs 弯曲：扭转是「剪切问题」，弯曲是「正应力问题」。** 但公式结构相似：应力=内力×距离/几何量。

### 六、检索能力检查 + MATLAB

1. 写出实心圆轴扭转最大切应力公式。
2. 为什么传动轴常用空心轴？

## 第16天：弯曲内力——剪力图与弯矩图

### 一、建立地图

```
材料力学
└── 弯曲 ← 今天开始（最重点！）
    ├── 内力：剪力V、弯矩M
    ├── 应力：正应力σ、切应力τ
    └── 变形：挠度、转角
```

### 二、统一抽象

**剪力与弯矩的统一理解**：

```
剪力V：截面一侧所有外力的代数和（使截面两侧相对滑移）
弯矩M：截面一侧所有外力对截面形心力矩的代数和（使梁弯曲）

符号规定（重要！）：
  剪力：左上右下为正
  弯矩：使梁下凸为正（简支梁受向下力，弯矩为正）
```

**微分关系**（画内力图的利器！）：
```
dV/dx = q(x)       剪力对x的导数 = 分布载荷集度
dM/dx = V(x)       弯矩对x的导数 = 剪力
d²M/dx² = q(x)     弯矩二阶导数 = 分布载荷
```

### 三、核心矛盾

> **弯矩图为什么重要？**
> - 最大弯矩位置 → 最危险截面 → 强度设计依据
> - A类题中，梁的优化设计直接依赖弯矩图

### 四、因果链/易混点

**画剪力图弯矩图的步骤**：
```
1. 求支座反力
2. 分段（集中力、力偶处为分段点）
3. 列各段V(x)和M(x)方程
4. 利用微分关系检验图形合理性
   q=0 → V=常数 → M=线性
   q=常数 → V=线性 → M=抛物线
```

**易混点**：
- 集中力处：剪力突变，弯矩连续但有尖角
- 集中力偶处：弯矩突变，剪力连续
- 均布载荷段：弯矩图为抛物线，凸向与q方向一致

### 五、压缩提炼

> **弯矩图的记忆口诀**：
> - 无载段：剪力平，弯矩斜
> - 均载段：剪力斜，弯矩抛
> - 集中力：剪力突变，弯矩折角
> - 力偶处：剪力不变，弯矩突变

### 六、检索能力检查 + MATLAB

1. 简支梁受均布载荷，画剪力图和弯矩图，指出最大弯矩位置和大小。
2. 微分关系 dM/dx=V 的物理意义是什么？

```matlab

└── 变形
```

### 二、统一抽象

```
弯曲正应力公式：
  σ = My / I
  M：弯矩  y：到中性轴距离  I：截面惯性矩

关键概念：
  中性轴：σ=0的那一层（过形心）
  最远点应力最大：σmax = M / W
  W = I / y_max：抗弯截面系数

### 三、核心矛盾

> **为什么工字梁比矩形梁好？**
> - 弯曲正应力在中性轴处为零，在上下边缘最大

**易混点**：
- I（惯性矩，弯曲用）vs Iₚ（极惯性矩，扭转用）
- 矩形截面：I = bh³/12, W = bh²/6
- 圆形截面：I = πd⁴/64, W = πd³/32

### 五、压缩提炼

> **弯曲应力本质：弯矩使梁的一侧受拉、另一侧受压，中性轴处应力为零。**

## 第18天：弯曲变形——挠度与转角

### 一、建立地图

```
弯曲
├── 内力 ✓
├── 应力 ✓
└── 变形 ← 今天
```

### 二、统一抽象

```
弯曲变形的基本方程：
  EI y'' = M(x)    （挠曲线微分方程）

  y：挠度（向下为正）
  θ = y'：转角
  EI：抗弯刚度

求解方法：
  积分法：两次积分 → y(x)
  叠加法：查表叠加（工程常用）
  有限元：复杂梁（MATLAB实现）
```

### 三、核心矛盾

> **强度 vs 刚度**
> - 强度：不断裂（应力条件）
> - 刚度：变形不超标（挠度条件：y_max ≤ [y]）
> - 很多梁强度够了但刚度不够（细长梁）

### 四、因果链/易混点

**因果链**：
```
载荷 → 弯矩M(x) → EIy''=M(x) → 两次积分 → y(x), θ(x)
  → 刚度条件：y_max ≤ [y]
```

**常见梁的变形公式**（A类题必查）：
```
简支梁均布载荷：
  y_max = 5qL⁴/(384EI), 位于跨中
  θ_A = θ_B = qL³/(24EI)

悬臂梁端部集中力：
  y_max = PL³/(3EI), 位于自由端
  θ_max = PL²/(2EI), 位于自由端
```

### 五、压缩提炼

> **变形 = 载荷 × Lⁿ / (EI × 系数)**。增大EI或减小跨度，可有效减小变形。

I = 1e7;     % mm⁴（惯性矩）
L = 6000;    % mm
P = 50e3;    % N，跨中集中力

n = 200;                    % 离散点数
x = linspace(0, L, n);
dx = L/(n-1);

% 弯矩分布
M = zeros(1,n);
for i = 1:n
    if x(i) <= L/2
        M(i) = P*x(i)/2;
    else
        M(i) = P*(L-x(i))/2;
    end
end

% 数值积分求挠度：EI y'' = M(x)
% 两次积分
y_double_prime = M / (E*I);

% 积分1次 → 转角θ
theta = cumtrapz(x, y_double_prime);
% 边界条件调整：简支梁 θ(0)≠0，需要用对称性
theta = theta - (theta(end)-theta(1))/2;

% 积分2次 → 挠度y

% 边界条件：y(0)=0, y(L)=0
y = y - y(1) - (y(end)-y(1))*(x/L);

y_analytical = P*x .*(3*L^2 - 4*x.^2) / (48*E*I);
y_analytical(x > L/2) = P*(L-x(x>L/2)) .*(3*L^2 - 4*(L-x(x>L/2)).^2) / (48*E*I);

fprintf('数值解最大挠度 = %.3f mm\n', -min(y));
fprintf('解析解最大挠度 = %.3f mm\n', max(y_analytical));

## 第19天：应力状态与强度理论

### 一、建立地图

```
材料力学深化
└── 应力状态与强度理论 ← 今天
    → 解决「复杂受力下，材料什么时候破坏？」
```

### 二、统一抽象

```
一点处的应力状态：
  用微元体上的应力分量描述
  2D: σx, σy, τxy

主应力：切应力为零的面上的正应力
  σ₁ ≥ σ₂ ≥ σ₃（按大小排列）

四种强度理论：
  1. 最大拉应力理论：σ₁ ≤ [σ]         （脆性材料）
  2. 最大伸长线应变理论：σ₁-μ(σ₂+σ₃) ≤ [σ]
  3. 最大切应力理论：σ₁-σ₃ ≤ [σ]      （塑性材料，偏安全）
  4. 形状改变比能理论：√(½[(σ₁-σ₂)²+(σ₂-σ₃)²+(σ₃-σ₁)²]) ≤ [σ]  （塑性材料，最常用）
```

### 三、核心矛盾

> **不同材料，破坏机理不同：**
> - 脆性材料（铸铁）：断裂 → 用第一强度理论
> - 塑性材料（钢）：屈服 → 用第三或第四强度理论
> - 选错理论 = 设计不合理

### 四、因果链/易混点

**因果链**：
```
受力分析 → 某点应力状态 → 求主应力 → 选强度理论 → 校核
```

**易混点**：
- 第三理论（最大切应力）偏保守，第四理论更精确但复杂
- 工程中钢用第四理论，铸铁用第一理论

### 五、压缩提炼

> **强度理论的本质：把复杂的多向应力状态，折算成单向的「等效应力」，与许用应力比较。**

### 六、检索能力检查 + MATLAB

1. 四种强度理论分别适用于什么材料？

## 第20天：压杆稳定

### 一、建立地图

```
材料力学
└── 压杆稳定 ← 今天（A类题高频！结构失稳）
```

### 二、统一抽象

```
欧拉公式：
  F_cr = π²EI / (μL)²

  F_cr：临界压力（失稳载荷）
  μ：长度系数（取决于约束）
    两端铰支：μ=1
    一端固定一端自由：μ=2
    两端固定：μ=0.5
    一端固定一端铰支：μ=0.7
  L：杆长
  I：截面惯性矩（取最小值！）

柔度：λ = μL / i
  i = √(I/A)：截面回转半径
  λ越大 → 越细长 → 越容易失稳
```

### 三、核心矛盾

> **强度问题 vs 稳定问题**
> - 短粗杆：强度控制（先断，不失稳）
> - 细长杆：稳定控制（还没断就失稳了！）
> - **细长杆的失稳往往在远低于强度极限的载荷下发生——这是工程事故的常见原因**

### 四、因果链/易混点

**因果链**：
```
压力F → 杆有微小弯曲趋势
  → F<F_cr：弯曲恢复（稳定平衡）
  → F=F_cr：弯曲保持（临界状态）
  → F>F_cr：弯曲急剧增大（失稳！）
```

**易混点**：
- I取最小惯性矩（失稳在最小刚度方向发生）
- λ>λ_p（比例极限柔度）才能用欧拉公式
- 中柔度杆用经验公式（直线公式），小柔度杆用强度条件

### 五、压缩提炼

> **压杆稳定的本质：轴向压力使杆的弯曲刚度「失效」。** 压力越大，有效抗弯刚度越小，到临界值时彻底丧失。

## 第22天：流体力学地图 + 流体性质

### 一、建立地图

└── 相似与量纲分析：模型实验
```

### 二、统一抽象

**流体 vs 固体的根本区别**：

```
固体：有固定形状，抵抗剪切变形
流体：没有固定形状，不能抵抗剪切（静止时！）
      → 流体静止时只有正应力（压力），没有切应力
      → 流体运动时才出现粘性切应力
```

**流体的三个基本物性**：
```
密度 ρ：单位体积质量（kg/m³），水≈1000，空气≈1.2
粘度 μ：抵抗变形的能力（Pa·s），水≈10⁻³，空气≈1.8×10⁻⁵
压缩性：液体几乎不可压，气体可压缩
  → 工程简化：液体=不可压流体，低速气体≈不可压
```

### 三、核心矛盾

> **粘性 vs 理想流体**
> - 真实流体都有粘性（如水的粘度虽小但不为零）
> - 理想流体（μ=0）是数学抽象，很多问题必须先忽略粘性才能求解
> - **但粘性正是阻力、能耗、湍流等的根源——不能永远忽略**

### 四、因果链/易混点

**因果链**：
```
流体物性
  ├── 密度 → 惯性力 → 动量
  ├── 粘度 → 粘性力 → 阻力、边界层
  └── 压缩性 → 声速 → 可压/不可压判断
```

**易混点**：
- 动力粘度μ vs 运动粘度ν=μ/ρ：μ是物性，ν是μ/ρ，更方便计算
- 理想流体≠理想气体（前者忽略粘性，后者是状态方程p=ρRT）

### 五、压缩提炼

> **流体力学的起点：流体不能抵抗剪切→静止时只有压力→运动时粘性产生切应力。**

### 二、统一抽象

```
静力学基本方程：
  dp/dz = -ρg     或     p = p₀ + ρgh

  p₀：液面压力
  h：距液面深度
  ρgh：液柱产生的压力

**阿基米德原理**：浮力 = 排开流体的重量 = ρgV_displaced

### 三、核心矛盾

> **绝对压力 vs 相对压力（表压）**
> - 绝对压力：以完全真空为零点
> - 表压：以大气压为零点（工程常用）
> - p_abs = p_atm + p_gauge
> - **A类题中务必明确用哪种！**

### 四、因果链/易混点

**因果链**：
```
重力场 → 静压力随深度线性增加 → p = p₀ + ρgh
  → 作用于壁面的力 F = ∫p dA
  → 矩形壁：F = ρgbh²/2（三角形分布积分）
```

**易混点**：
- 压力只与深度有关，与容器形状无关（静水奇观）
- 浮力作用点（浮心）≠ 重心位置，两者不重合时产生恢复力矩

### 五、压缩提炼

> **静力学核心：p=ρgh，压力只看深度。** 力=压力×面积，注意压力分布不均匀时的积分。

- 流线≠迹线（非定常时不同）
- 连续性方程是运动学方程（不含力），适用于任何流体
- 不可压条件：马赫数Ma<0.3时近似成立

## 第25天：伯努利方程——A类题核心工具

### 一、建立地图

```
流体动力学
└── 理想流体 ← 今天
    └── 伯努利方程（能量守恒）
```

### 二、统一抽象

```
伯努利方程（沿流线，定常，不可压，理想流体）：

  p/ρg + v²/(2g) + z = 常数

  或  p + ½ρv² + ρgz = 常数

三项的物理意义：
  p          → 压力能（静压）
  ½ρv²      → 动能（动压）
  ρgz       → 位能（势能）

统一理解：伯努利方程就是流体的能量守恒！
```

### 三、核心矛盾

> **伯努利方程的四大前提（一个不满足就不能用！）**：
> 1. 理想流体（无粘性）——有摩擦时需修正
> 2. 定常流动——非定常时需加∂φ/∂t项
> 3. 不可压缩——高速气体不适用
> 4. 沿同一流线——不同流线常数不同

### 四、因果链/易混点

**因果链**：
```
沿流线的能量守恒
  → p₁ + ½ρv₁² + ρgz₁ = p₂ + ½ρv₂² + ρgz₂
  → 截面小→v大→p小（文丘里效应）
  → 高处→z大→p小或v小
```

**应用场景**（A类题高频）：
```
1. 文丘里管：测流量 Q = A₂√[2(p₁-p₂)/(ρ(A₁²/A₂²-1))]
2. 皮托管：测流速 v = √(2Δp/ρ)
3. 大容器出流：v = √(2gh)（托里拆利公式）

## 第27天：粘性流动与N-S方程

### 一、建立地图

└── 粘性流体 ← 今天
    └── N-S方程（流体力学最高方程）
```

### 二、统一抽象

```
N-S方程（不可压缩）：

  ρ(∂v/∂t + v·∇v) = -∇p + μ∇²v + ρg

  左边：惯性力（加速度项）
  右边：压力梯度 + 粘性力 + 重力

理解：
  N-S方程 = 牛顿第二定律的流体版本
  力 = 质量 × 加速度 → 逐项对应

简化路径：
  N-S方程 → 忽略粘性 → 欧拉方程 → 积分 → 伯努利方程
  N-S方程 → 定常+简化 → 各种解析解
```

### 三、核心矛盾

> **N-S方程是已知的，但一般无法解析求解！**
> - 这是流体力学最大的困难
> - A类题中：用数值方法（有限差分、有限元）或简化模型
> - **雷诺数Re是决定流动特征的关键参数**

```
Re = ρvL/μ = vL/ν

Re << 1：粘性主导（蠕流/Stokes流），N-S可简化
Re ~ 1-2000：层流，有解析可能
Re >> 2000：湍流，必须数值求解或实验
```

### 四、因果链/易混点

**因果链**：
```
Re = 惯性力/粘性力
  → Re小：粘性主导，流动规则（层流）
  → Re大：惯性主导，流动混乱（湍流）
  → 临界Re：层流→湍流的转捩点
     管流：Re_cr ≈ 2300
```

**易混点**：
- Re无量纲，但与特征长度L的选择有关（管流L=直径，外流L=物体长度）
- N-S方程没有普适解析解≠不能解——特殊情况可以

### 五、压缩提炼

> **N-S方程是流体力学的「终极方程」，但也是「终极难题」。** 工程中靠Re判断简化方向：低Re忽略惯性，高Re用湍流模型。

## 第28天：管道流动——摩阻与水头损失

### 一、建立地图

```
粘性流体应用
└── 管道流动 ← 今天（A类题高频！）
    ├── 层流解析解
    ├── 湍流经验公式
    └── 水头损失计算
```

### 二、统一抽象

```
修正的伯努利方程（含水头损失）：

  p₁/ρg + v₁²/(2g) + z₁ = p₂/ρg + v₂²/(2g) + z₂ + h_f

  h_f：水头损失（能量被摩擦消耗）

  f：沿程阻力系数（达西摩擦因子）
  L：管长，D：管径

  层流：f = 64/Re
  湍流：查Moody图或用Colebrook公式
    1/√f = -2log(ε/(3.7D) + 2.51/(Re√f))
```

### 三、核心矛盾

> **层流有精确解，湍流只有经验公式。**
> - 层流：f=64/Re，简单精确
> - 湍流：Moody图/Colebrook公式，需要迭代求解
> - **A类题中管道问题，判断流态是第一步！**

### 四、因果链/易混点

**因果链**：
```
管流问题求解流程：
  1. 计算Re → 判定层流/湍流
  2. 选f公式：层流f=64/Re，湍流查Moody/Colebrook
  3. 计算h_f = fLv²/(2gD)
  4. 代入修正伯努利方程
```

**易混点**：
- 沿程损失（摩擦）vs 局部损失（阀门、弯头）：h_local = K·v²/(2g)
- 总损失 = 沿程损失之和 + 局部损失之和
- Colebrook公式需要迭代——MATLAB中用fsolve

### 一、建立地图

└── 量纲分析与相似律 ← 今天
    → 模型实验设计的理论基础
    → A类题中的参数归约工具
```

### 二、统一抽象

```
白金汉π定理：
  如果某物理问题有n个变量，涉及k个基本量纲，
  则可组成(n-k)个无量纲π组。

关键无量纲数（A类题必须知道的）：
  Re = ρvL/μ       雷诺数（惯性/粘性）
  Fr = v/√(gL)      弗劳德数（惯性/重力）
  Eu = p/(ρv²)      欧拉数（压力/惯性）
  Ma = v/c          马赫数（流速/声速）
  We = ρv²L/σ       韦伯数（惯性/表面张力）
```

### 三、核心矛盾

> **实物太大/太贵，需要做缩尺模型实验。**
> 但模型和实物必须「相似」——几何相似+运动相似+动力相似。
> **动力相似要求：所有有关的π数必须相等。**
> 实际中往往只能满足主要π数（如管流保证Re相等即可）。

### 四、因果链/易混点

**因果链**：
```
确定影响因素 → 列出所有变量 → 选基本量纲(M,L,T)
  → 用π定理组合 → 得到无量纲关系
  → f(π₁, π₂, ...) = 0
  → 实验只需求无量纲关系，变量数大幅减少
```

**易混点**：
- 不同问题起决定作用的无量纲数不同：
  - 管流/阻力问题 → Re
  - 船舶/明渠 → Fr
  - 高速气体 → Ma
  - 毛细管/液滴 → We

### 五、压缩提炼

> **量纲分析的本质：物理规律不依赖于单位的选择。** 用无量纲数描述问题，变量数减少，实验量减少，结果通用。

### 六、A类题综合实战：城市供水管网水力分析

```matlab
%% 第30天 A类题综合实战：供水管网水力分析
% 问题：从水塔到用户，经过变径管道，求各节点压力和流量
% 涉及：连续性方程 + 修正伯努利方程 + 摩阻计算

rho = 1000; g = 9.81; mu = 1e-3;

% 管网数据：3段管道串联
pipes = [...
    0.15, 500, 0.0002;   % 段1: D=150mm, L=500m, ε=0.2mm
    0.10, 300, 0.0002;   % 段2: D=100mm, L=300m
    0.08, 200, 0.0002];  % 段3: D=80mm,  L=200m

% 水塔水位
H_tank = 30;   % m

% 迭代求解流量（给定水头差，求Q）
% 总水头损失 = 水塔水位
% h_f1 + h_f2 + h_f3 = H_tank

Q_guess = 0.005;  % 初始猜测 m³/s

for outer = 1:30
    h_f_total = 0;
    for i = 1:3
        D = pipes(i,1); L = pipes(i,2); eps = pipes(i,3);
        A = pi*D^2/4;
        v = Q_guess / A;
        Re = rho*v*D/mu;
        
        if Re < 2300
            f = 64/Re;
        else
            f = 0.02;
            for iter = 1:50
                f_new = 1/(-2*log10(eps/(3.7*D)+2.51/(Re*sqrt(f))))^2;
                if abs(f_new-f)<1e-8, break; end
                f = f_new;
            end
        end
        h_f_total = h_f_total + f*L/D*v^2/(2*g);
    end
    
    % 调整Q：h_f_total应该等于H_tank
    % h_f ∝ Q²（近似）
    Q_guess = Q_guess * sqrt(H_tank / h_f_total);
end

fprintf('=== 供水管网计算结果 ===\n');
fprintf('总流量 Q = %.4f m³/s = %.1f L/s\n', Q_guess, Q_guess*1e3);
fprintf('水塔水位 = %.1f m\n', H_tank);

% 各段详细信息
p_outlet = rho*g*H_tank;  % 起点压力（表压）
for i = 1:3
    D = pipes(i,1); L = pipes(i,2); eps = pipes(i,3);
    A = pi*D^2/4;
    v = Q_guess / A;
    Re = rho*v*D/mu;
    if Re < 2300
        f = 64/Re;
        state = '层流';
    else
        f = 0.02;
        for iter = 1:50
            f_new = 1/(-2*log10(eps/(3.7*D)+2.51/(Re*sqrt(f))))^2;
            if abs(f_new-f)<1e-8, break; end
            f = f_new;
        end
        state = '湍流';
    end
    h_f = f*L/D*v^2/(2*g);
    p_outlet = p_outlet - rho*g*h_f;
    
    fprintf('\n段%d: D=%.0fmm, L=%.0fm\n', i, D*1e3, L);
    fprintf('  v=%.2fm/s, Re=%.0f(%s), f=%.4f\n', v, Re, state, f);
    fprintf('  水头损失=%.2fm, 出口表压=%.1fkPa\n', h_f, p_outlet/1e3);
end

% 灵敏度分析（A类题加分项）：水塔高度变化对流量影响
H_range = 10:1:50;
Q_result = zeros(size(H_range));
for k = 1:length(H_range)
    Q_g = 0.005;
    for outer = 1:30
        hf_t = 0;
        for i = 1:3
            D = pipes(i,1); L = pipes(i,2); eps = pipes(i,3);
            A = pi*D^2/4; v = Q_g/A;
            Re = rho*v*D/mu;
            if Re<2300, f=64/Re; else, f=0.02;
                for iter=1:50, fn=1/(-2*log10(eps/(3.7*D)+2.51/(Re*sqrt(f))))^2;
                    if abs(fn-f)<1e-8,break;end; f=fn; end; end
            hf_t = hf_t + f*L/D*v^2/(2*g);
        end
        Q_g = Q_g * sqrt(H_range(k)/hf_t);
    end
    Q_result(k) = Q_g;
end

figure; plot(H_range, Q_result*1e3, 'r-', 'LineWidth', 2);
xlabel('水塔水位 (m)'); ylabel('流量 (L/s)');
title('供水管网灵敏度分析'); grid on;
```
